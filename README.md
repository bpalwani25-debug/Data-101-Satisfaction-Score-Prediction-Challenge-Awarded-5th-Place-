The data contains roughly 10,000 entries and consists of data 
about test takers in  a test taking faciity. The columns are testId, 
testDate, age, personality, testScore, and satisfactionScore, with 
satisfactionScore being the target attribute.  The satisfactionScore 
attribute returns the mean  score of satisfaction questionaire which 
is filled by each test taker after they take a test.  Training data 
can be found in folder PredictionChallenge-Data under JBTrain.csv. 
Testing data - without target attribute of course, is in 
JBTest-sttudents..csv.  Finally there is submission.csv file which 
you need to fill with your predictions.
